## What can you build with Xata?

Xata is a versatile platform that provides a range of resources to help you start building your data project. In addition to our getting started [framework apps](/docs/getting-started/intro), you can also craft solutions that are tailored to your projects. We've gathered practical examples to show you how Xata works with different technologies and services. Explore our example apps to gain insights into how Xata can help in your development projects.

The following material is available in the `apps` folder of the Examples repository:

| Example app type      | Prefix             | Description                                                                                                                                                                                                             |
| --------------------- | ------------------ | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Getting started guide | `getting-started-` | This includes comprehensive walkthrough tutorial that guide you through the process of using Xata with an application framework.                                                                                        |
| Sample                | `sample-`          | This includes examples of real-world integrations with Xata, as well as solutions suitable for production-grade applications. Additionally, it provides insights into reverse engineering and strategic considerations. |
| Starter               | `starter-`         | This category is intended for starter examples, but it seems to be missing a specific description. You can add a description here if needed.                                                                            |

### Getting started guides

Getting started guides provide step-by-step instructions for using Xata with a popular framework. For more in-depth information, you can see to our comprehensive [walkthrough documentation]() or explore the GitHub repository.

Our samples provide step-by-step guidance for using Xata in your projects. Whether it's for user authentication or handling image and file uploads and storage, these examples offer a clear solution.

- **[Prune your follows](/docs/examples/prune-your-follows):** Develop a web app with Xata, GatsbyJS, NextAuth and Tailwind to find accounts on Twitter to unfollow.

### Starter applications

Xata's starter applications serve as ready-made project foundations. These GitHub repositories are easy to clone or fork, and give you a solid starting point for your projects. Each starter includes a detailed README explaining how to set up the application.

Explore these starter options:

- [Next.js](https://github.com/vercel/next.js/tree/canary/examples/with-xata)
- [Remix + Netlify](https://github.com/xataio/examples/tree/main/apps/starter-remix-netlify)
- [Vercel Serverless Functions](https://github.com/xataio/examples/tree/main/apps/starter-vercel-serverless-functions)

You can use a starter as-is or customize it to fit your project's unique needs.

For a comprehensive list of examples and starters, visit our GitHub [examples repository](https://github.com/xataio/examples/tree/main/apps).
